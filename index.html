Watch the messages about gravity.js fall down and bounce when you visit the page!

gravity.js is a simple library for adding gravity effects to web elements.
It uses Matter.js internally for physics simulation.
With gravity.js, you can make any DOM element fall and interact with the page.
Perfect for creating fun and interactive animations on your website.
To start, simply include the library and add a class to elements you want to fall.
Try it out and see the effect immediately when the page loads!
